# سجل وإستراتيجية إصدارات المنصة (RELEASE_HISTORY.md) - Nebras ERP

يوثق هذا السجل التطور التاريخي للإصدارات وتطلعات النشر المستقبلي للمنصة.

---

## 1. جدول الإصدارات التراكمي (Release Roadmap & Version History)

### [v0.9.0] - 2026-07-02 (الإصدار الأخير النشط)
- **العنوان:** ERP Workspace & Navigation App Shell.
- **الميزات:**
  - بناء شجرة التنقل الكاملة وتصفيتها بالصلاحيات (RBAC).
  - ربط ودجات الإحصائيات حية من قاعدة البيانات وعرض أحدث النشاطات الأمنية والبحث المركزي (CTRL + K).
- **حالة الترقية:** متوافقة تماماً.

### [v0.8.5] - 2026-07-02
- **العنوان:** Platform Audit, Gap Analysis & Central Handler.
- **الميزات:**
  - دمج الـ `BaseAppException` والوراثة الموحدة للـ `PlatformBaseViewSet` من `BaseCRUDViewSet`.
  - حساب مؤشرات الأداء (RAM, CPU, Database Size) حية من الخادم.

### [v0.8.0] - 2026-07-02
- **العنوان:** Shared Foundation Library.
- **الميزات:** بناء مكتبة `apps/shared` المشتركة وتوحيد كلاسات الـ DDD وأنماط النطاق الأساسية.

### [v0.7.0] - 2026-07-02
- **العنوان:** Platform Kernel & Notification Center.
- **الميزات:** بناء الـ Event Bus المشترك، ناقل الأحداث، مركز التنبيهات، والرفع الآمن للملفات.

### [v0.6.0] - 2026-07-02
- **العنوان:** Student Lifecycle Management.
- **الميزات:** بناء دورة حياة الطالب كـ Aggregate Root ومولد الأرقام التسلسلية.

### [v0.5.0] - 2026-07-02
- **العنوان:** Admissions & Workflow Engine.
- **الميزات:** بناء محرك مسارات العمل للقبول والتسجيل والمقابلات الشخصية.

### [v0.1.0] - 2026-07-01
- **العنوان:** Architectural Foundation & Security (IAM).
- **الميزات:** تأسيس قاعدة البيانات الموحدة ونظام عزل المستأجرين والـ Middleware ونظام المصادقة JWT.
