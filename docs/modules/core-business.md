# نواة الأعمال المشتركة للمنصة (Nebras Core Business Platform - NCBP)

توفر هذه الوثيقة دليلاً مرجعياً ومعمارياً شاملاً لطبقة الأعمال المشتركة **NCBP** والتي تعتبر المكون الأساسي لجميع موديولات نظام Nebras ERP الحالية والمستقبلية.

---

## 1. الهيكل المعماري والطبقات (NCBP Architecture)

يتبع التطبيق `core_business` نمط التطوير المعياري القياسي (Clean Architecture):

- **Domain Models:** نماذج الكيانات الأساسية المتوافقة مع الـ Multi-Tenancy والحذف اللطيف.
- **Application Services:** خدمات الاستيراد والتصدير والمولد الموحد للأرقام المرجعية.
- **Interfaces APIs:** الـ ViewSets وسجلات الـ Endpoints للربط مع الفرونت إند وتطبيق الجوال.

---

## 2. المكونات المشتركة للواجهة الأمامية (Reusable Angular Components)

تم تطوير 3 مكونات مركزية كأدوات Standalone وتعمل بالـ Signals جاهزة للاستدعاء المباشر في أي صفحة:

1. **app-attachment-viewer:** لعرض الملفات المرفقة للكيانات وتنزيلها وحذفها.
2. **app-comment-widget:** لإدراج التعليقات والملاحظات والتغذية الراجعة.
3. **app-timeline-widget:** لعرض الخط الزمني التفصيلي وسجل الحركات والتدقيق الأمني.

---

## 3. مصفوفة الصلاحيات (Permissions Matrix)

تخضع جميع واجهات الـ API لنظام الصلاحيات الديناميكي والأدوار (RBAC):

- `attachments.*`: صلاحيات إدارة وعرض المرفقات.
- `comments.*`: صلاحيات إضافة وحذف وحل التعليقات.
- `lookup.*`: صلاحيات إدارة قواميس البيانات الثابتة.
- `timeline.*`: صلاحيات استعراض الخط الزمني التاريخي.
